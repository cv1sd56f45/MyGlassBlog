# Hello World 🌍

欢迎来到我的个人博客！

这是使用 **Next.js 15** 搭建的毛玻璃（Glassmorphism）风格博客系统。

## 特色功能

- ✨ 毛玻璃 UI 设计
- 🌙 深色模式支持
- 📱 响应式布局
- 📝 Markdown 文章支持
- 💬 说说 / 时间线 / 照片墙

## 如何发布文章

只需在 `posts/` 目录下创建 `.md` 文件，并在文件开头添加 YAML frontmatter：

```yaml
---
title: "文章标题"
date: "2026-06-06"
category: "分类"
description: "文章描述"
cover: "https://example.com/cover.jpg"
---
```

## 技术栈

- **Next.js 15** - React 框架
- **Tailwind CSS 4** - 样式
- **Framer Motion** - 动画
- **next-themes** - 主题切换
- **gray-matter + remark** - Markdown 解析
